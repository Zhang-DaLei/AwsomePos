# 前言

>使用vue模仿的一个初级点餐系统，该系统目前只有点餐功能、添加商品、删除商品，结账功能，后续完善其他功能。

## 代码使用

``` bash

1.npm install（如果太慢建议cnpm）

2.npm run dev
```
